# Go Long Environment

A minimal Go workspace scaffolded for development.

## Quick start

1. Verify Go is installed:
   ```sh
go version
```
2. Run the project:
   ```sh
go run .
```
3. Build the binary:
   ```sh
go build .
```

## Workspace structure

- `go.mod` — module file
- `main.go` — starter application
- `.vscode/tasks.json` — editor tasks for running and building
